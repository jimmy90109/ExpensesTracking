# ExpensesTracking 記帳APP
### 國立台北教育大學 110學年度 嵌入式系統軟體設計 期末專題

* 目的與優點  
是簡單介面、操作容易的記帳APP，  
加上目前手機都配備生物辨識功能，新增了需要經過生物辨識才能進到APP裡的門檻，  
又為使用者加上一級安全性。
